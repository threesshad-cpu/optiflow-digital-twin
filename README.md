# optiflow-digital-twin
## How to Run Locally
bash
```
git clone https://github.com/threesshad-cpu/optiflow-digital-twin.git
cd project
npm install
npm run dev
```
### Backend
bash
```
cd server
npm install
npm start
```
### Frontend
bash
```
cd client
npm install
npm start
```
Runs on:
* **Frontend: http://localhost:3000**
* **Backend: http://localhost:3001**

Node version (e.g., Node 18)

Any env variables
